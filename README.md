Generate template files(babelrc, eslint, readme, etc...) from the command line

## Install

```cli
npm install -g keza
```

or

```cli
yarn global add keza
```

## Using Keza CLI

#### Command Line

```cli
$ keza
```

## Options:

- `--help`: display helper screen.
